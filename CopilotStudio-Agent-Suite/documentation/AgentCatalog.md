# Copilot Studio Agent Catalog

A comprehensive overview of all agents in the Copilot Studio suite, including their core skills, primary purpose, and unique value.

---

## 1. Entra Security Manager
- Manages Entra ID security policies and identity governance
- Assists with access reviews, compliance, and risk monitoring
- Provides guidance on conditional access and privileged identity management

## 2. Entra Security Ops
- Monitors and responds to security incidents in Entra environments
- Supports threat detection, investigation, and remediation workflows
- Integrates with security operations tools for real-time alerts

## 3. Dynamics Resource Manager
- Manages Dynamics 365 resources and user entitlements
- Automates environment provisioning and license assignments
- Provides insights into usage, health, and compliance of Dynamics assets

## 4. Azure Subscriptions
- Manages Azure subscription lifecycle and access controls
- Tracks resource usage, cost, and compliance across subscriptions
- Supports automated provisioning and deprovisioning of Azure resources

## 5. Resource Inventory
- Maintains a unified inventory of cloud and on-premises assets
- Enables asset discovery, classification, and lifecycle tracking
- Supports reporting for audits, compliance, and capacity planning

## 6. Internal Compliance
- Monitors adherence to internal and regulatory compliance standards
- Automates evidence collection and compliance reporting
- Flags policy violations and recommends remediation actions

## 7. Internal Security
- Detects and responds to internal security threats and anomalies
- Monitors privileged access and sensitive operations
- Provides actionable security recommendations for enterprise environments

## 8. Power Platform
- Manages Power Platform environments, apps, and flows
- Automates governance, security, and lifecycle management
- Provides analytics on usage, performance, and compliance

## 9. Dataverse
- Manages Dataverse data, schema, and security
- Supports data integration, migration, and quality monitoring
- Enables advanced search, reporting, and access control

## 10. Kusto
- Facilitates Kusto (Azure Data Explorer) query and analytics operations
- Automates data ingestion, transformation, and visualization
- Provides insights for monitoring, troubleshooting, and optimization

---

Each agent is designed for seamless integration, robust automation, and domain-specific expertise to accelerate your business processes and ensure compliance, security, and operational excellence.
