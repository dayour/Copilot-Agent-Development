# Clothing

Showcase library of Copilot Studio agents for the Clothing vertical. Each agent includes a runbook, conversation template, and solution definition file ready to import into your Power Platform environment.

## Agents

| Agent | Description |
|-------|-------------|
| [Power Analysis](./agents/power-analysis/) | A business intelligence agent providing conversational access to sales performance, inventory analytics, trend insights, customer metrics, and operational KPIs |
