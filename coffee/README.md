# Coffee

Showcase library of Copilot Studio agents for the Coffee vertical. Each agent includes a runbook, conversation template, and solution definition file ready to import into your Power Platform environment.

## Agents

| Agent | Description |
|-------|-------------|
| [Virtual Coach](./agents/virtual-coach/) | An always-available coach for baristas and staff covering recipes, HR policies, shift handover, and performance tips |
