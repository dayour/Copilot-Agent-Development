# Pipeline Analytics and Deal Coaching Agent (Tech)

## Overview

Pipeline Analytics and Deal Coaching is a Copilot Studio agent for enterprise sales organizations using Salesforce CRM. It is designed for sales managers, account executives, and sales operations teams who need real-time pipeline visibility, deal health intelligence, AI-driven coaching, and forecast accuracy.

- Agent Name: Pipeline Analytics and Deal Coaching
- Vertical: Tech
- Primary Users:
  - Sales Managers
  - Account Executives
  - Sales Operations Analysts
  - Sales Directors

## Business Goals

1. Surface real-time pipeline health and weighted forecast from Salesforce without leaving Microsoft Teams.
2. Identify at-risk deals early using activity recency, stakeholder engagement, and competitor signal analysis.
3. Deliver data-driven coaching nudges that correlate deal behaviors with historical close-rate patterns.
4. Produce manager-level forecast roll-ups by rep with best-case and weighted scenario projections.
5. Explain win and loss patterns from closed opportunities to improve future sales execution.

## Channel Strategy

### Internal Channel: Microsoft Teams

- Audience: Sales managers, AEs, sales operations
- Auth: Microsoft Entra ID (Azure AD SSO)
- Data Access: Full Salesforce CRM read access via Power Automate flows
- Use Cases:
  - Pipeline summary by stage with weighted forecast
  - Deal health check with stakeholder and competitor analysis
  - Deal coaching nudges based on activity patterns
  - Team forecast roll-up by rep for a given quarter
  - Win/loss analysis for closed opportunities

## Integration Architecture Summary

- Copilot Studio orchestrates conversation and topic routing in generative orchestration mode.
- Power Automate flows retrieve and aggregate Salesforce Opportunity, Task, Contact, and Account data.
- Historical stage-conversion rates are stored in Dataverse for weighted forecast calculations.
- Win/loss reason codes and competitor tags are pulled from closed Salesforce Opportunities.

## Design Notes

- Salesforce connector latency target: 2 to 6 seconds per query.
- Deal Coaching nudges are generated by comparing current deal behavior against historical patterns stored in Dataverse.
- All topics are internal-only; this agent does not support an external channel.
- Weighted forecast applies the stage probability percentage from Salesforce to each opportunity amount.
- Win/loss analysis queries Salesforce Opportunities with Stage = Closed Lost within the requested time window.
