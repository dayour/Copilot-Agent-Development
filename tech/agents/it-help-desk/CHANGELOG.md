All notable changes to the IT Help Desk agent will be documented in this file.

## [1.0.0] - 2026-03-02

### Added

- Initial release of the IT Help Desk agent for the Tech vertical.
- Password reset, hardware request, and general ITSM ticket creation topics.
- SharePoint knowledge source binding for IT KB articles.
- Power Automate flows for ITSM platform integration (ServiceNow / Jira).
- Microsoft Teams channel configuration with Azure AD authentication.
