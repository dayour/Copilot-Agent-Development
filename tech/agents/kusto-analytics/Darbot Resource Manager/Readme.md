# Kusto

A specialized conversational agent built with Microsoft Copilot Studio focused on Azure Data Explorer (Kusto), KQL development, and data analytics. This agent assists data engineers and analysts in comprehensive Kusto-based analytics solutions.

## Purpose

The Kusto agent is designed to help data teams with:

- **KQL Development**: Query language development and optimization
- **Data Analytics**: Advanced analytics and data exploration
- **Platform Management**: Cluster and database administration
- **Performance Optimization**: Query and cluster performance tuning
- **Integration**: Azure service integration and data pipelines
- **Visualization**: Data visualization and dashboard creation
